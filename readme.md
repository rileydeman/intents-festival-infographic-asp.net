# Intents Festival Infographic with ASP.net Core
A infographic about Intents Festival made with ASP.net Core.


## Installation / Running the Project (.NET 10)

This is an **ASP.NET Core Web** project targeting **.NET 10**.  
No solution file (`.sln`) is required.

### Prerequisites

- **.NET SDK 10.x**
    - Verify installation in the terminal:
      ```bash
      dotnet --version
      ```
- Any IDE that supports `.csproj` files:
    - Visual Studio 2022+
    - JetBrains Rider
    - VS Code (with C# Dev Kit)

### 1) Download the project

1. Go to the repository page
2. Click **Code → Download ZIP**
3. Extract the ZIP file to a local folder

### 2) Open the project in your IDE

Open the **`.csproj` file directly** (there is no `.sln` file):

- **Visual Studio**:  
  `File → Open → Project/Solution` → select the `.csproj`
- **Rider**:  
  `Open` → select the project folder or `.csproj`
- **VS Code**:  
  `File → Open Folder` → select the extracted folder

### 3) Run the application

Open a terminal in the project folder (the folder containing the `.csproj`) and run:

```bash
dotnet watch run
```
- The application will build automatically
- Your **default browser will open automatically**
- Hot reload is enabled by default

### 4) Access the app

Once running, the app will be available at:

- `http://localhost:<port>/`

The exact port is shown in the console output.

### Notes

- No solution file (`.sln`) is required
- File changes trigger automatic rebuilds and browser refresh
- Stop the app with `Ctrl + C`
